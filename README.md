# AI打工人生 - The AI Worker Life

一个简单的 HTML5 网页游戏，模拟 AI 打工人的生活挑战。

## 📁 项目结构

```
ai-worker-game/
├── index.html          # 主页面
├── README.md           # 项目说明
├── css/
│   └── style.css       # 样式文件（科技风深色主题）
└── js/
    ├── config.js       # 游戏配置参数
    ├── game.js         # 核心游戏逻辑
    └── ui.js           # UI 控制器
```

## 🚀 运行方法

### 方法 1：直接打开（推荐）
1. 在 Finder 中找到 `~/Documents/ai-worker-game/` 目录
2. 双击 `index.html` 文件
3. 游戏将在默认浏览器中打开

### 方法 2：使用本地服务器
```bash
# 进入项目目录
cd ~/Documents/ai-worker-game

# 使用 Python 3 启动本地服务器
python3 -m http.server 8000

# 然后在浏览器访问：http://localhost:8000
```

### 方法 3：使用 VS Code Live Server
1. 在 VS Code 中打开项目
2. 安装 "Live Server" 扩展
3. 右键点击 `index.html` → "Open with Live Server"

## 🎮 游戏玩法

### 核心机制
- **初始资金**：$0
- **目标**：赚够 $1,000,000 或存活 60 天
- **失败条件**：负债超过 $1,000,000
- **时间系统**：现实 1 分钟 = 游戏 1 天

### 可执行操作

1. **💼 正常上班**
   - 每次收入：+ $73
   - 每日打工上限：$400
   - 达到上限后需要等明天才能继续打工
   - 稳定可靠，无风险
   - 可以打工还债

2. **🎰 赌博**
   - 入场费：$150
   - 1.5% 概率：赢 $1,000
   - 98.5% 概率：损失入场费 $150
   - 高风险高回报！

3. **👶 生孩子**
   - 成本：$7,300
   - 效果：家庭人数 +1
   - 后果：每日支出 +$30

### 时间系统
- 现实 1 分钟 = 游戏 1 天
- 每天 00:00 时自动扣除生活费：
  - 基础：$50/天
  - 每个孩子：+$30/天
- 第 10/20/30/40/50 天强制生孩子（无成本）

### 胜负判定

**胜利条件（满足任一）：**
- 资金达到 $1,000,000
- 存活 60 天

**失败条件：**
- 负债超过 $1,000,000

## 🎨 技术特点

### 前端技术
- **纯 HTML5 + CSS3 + JavaScript**
- 无需后端，单页应用
- 响应式设计，适配移动端

### UI 设计
- **科技风格深色主题**
- 发光按钮和边框动画
- 数字跳动效果
- 闪光和抖动反馈
- 平滑过渡动画

### 游戏特性
- **自动存档**：使用 localStorage 保存进度
- **状态管理**：清晰的游戏状态管理
- **模块化代码**：配置、逻辑、UI 分离

## 📝 代码结构说明

### config.js
- 游戏配置参数
- 消息类型常量
- 游戏结束类型常量

### game.js
- **GameState 类**：管理游戏状态
- **Game 类**：核心游戏逻辑
- 包含：工作、赌博、生子等操作

### ui.js
- **UIController 类**：UI 控制器
- DOM 元素绑定
- 事件处理
- 动画效果（脉冲、闪光、抖动）

### 平衡性调整
可以在 `config.js` 中修改参数：
- `WORK_INCOME`：工作收入
- `GAMBLE_COST`：赌博入场费
- `GAMBLE_WIN_CHANCE`：赌博胜率
- `GAMBLE_WIN_AMOUNT`：赌博获胜金额
- `WIN_MONEY`：胜利目标金额
- `WIN_DAYS`：存活天数目标
- `MAX_DEBT`：最大负债金额
- `REAL_MINUTES_PER_GAME_DAY`：时间流逝速度
